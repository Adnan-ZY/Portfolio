# Muhammad Adnan's Portfolio

A modern, responsive portfolio website showcasing my work as a Software Engineer.

## Features

- **Responsive Design** - Fully mobile-optimized with Tailwind CSS
- **Dark/Light Theme** - Toggle between themes with smooth transitions
- **3D Project Slider** - Interactive Swiper carousel with coverflow effect
- **Skills Section** - Animated skill cards with Devicons
- **Smooth Animations** - Fade-in effects and hover interactions

## Tech Stack

- HTML5
- Tailwind CSS (CDN)
- JavaScript
- Swiper.js
- Font Awesome
- Devicons

## Sections

- **Home** - Hero with animated developer GIF and typewriter effect
- **Projects** - 3D coverflow slider showcasing project screenshots
- **Skills** - Visual display of technologies (C#, .NET, SQL Server, Angular, etc.)
- **About** - Education, experience, certifications, and personal info
- **Contact** - Contact cards with location, email, and phone
